# Project Customization Rules

- **Header Navigation:** DO NOT render navigation menus (such as "Tools", "Categories", "Blog", "About", "Pricing") in the header bar. Keep the header layout extremely clean, limited to the Logo on the left, Search bar in the center-right, and action icons (Favorites, Theme toggle, Sign in) on the right.
- **Git Push Policy:** Never run `git push` or deploy changes to remote git branches unless the user explicitly requests it.
